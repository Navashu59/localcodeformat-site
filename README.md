# localcodeformat.com

This directory is the active build workspace for `localcodeformat.com`.

## Important rules

- This site targets **new-site opportunity keywords**, not existing-site keyword expansion.
- The site must be built around **search intent + trust + topical authority**, not random tool sprawl.
- The site needs its **own GitHub repository** when the codebase is ready to split cleanly.

## First-wave launch focus

- json formatter
- json validator
- sql formatter
- html formatter
- diff checker

## Operational requirement

This site must launch with:

- sitemap
- robots.txt
- canonical tags
- internal hub links
- support guides
- visible privacy / local-only messaging

## Parallel rule

Work on this site does **not** replace operations on the existing portfolio sites.
